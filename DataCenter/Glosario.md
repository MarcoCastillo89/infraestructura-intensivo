# Glosario — Centro de Datos

## 1. Centro de datos
Lugar donde se concentran servidores, redes y sistemas para almacenar y procesar información.

## 2. Sala de servidores
Área donde están instalados los servidores y equipos principales de cómputo.

## 3. Sala de telecomunicaciones
Espacio donde se ubican switches, routers y cableado de red.

## 4. Sala eléctrica
Zona encargada de distribuir y respaldar la energía eléctrica del centro de datos.

## 5. Sala mecánica
Área donde están los sistemas de enfriamiento y ventilación.

## 6. Rack
Estructura metálica abierta donde se montan equipos de red y servidores.

## 7. Gabinete
Rack cerrado con puertas y paneles para proteger equipos y mejorar el flujo de aire.

## 8. Fila
Conjunto de racks o gabinetes colocados uno junto a otro.

## 9. Hot aisle
Pasillo donde los servidores expulsan el aire caliente.

## 10. Cold aisle
Pasillo por donde entra aire frío hacia los servidores.

## 11. Raised floor
Piso elevado que permite pasar cables y distribuir aire frío por debajo.

## 12. Plenum
Espacio usado para mover y distribuir aire dentro de la sala.

## 13. Servidor
Computadora especializada que brinda servicios o recursos a otros equipos.

## 14. Switch
Dispositivo que conecta equipos dentro de una red local.

## 15. Router
Equipo encargado de comunicar distintas redes y dirigir tráfico de internet.

## 16. Patch panel
Panel donde terminan y se organizan los cables de red.

## 17. Cableado estructurado
Sistema organizado de cables para red y telecomunicaciones.

## 18. Fibra óptica
Cable que transmite información usando luz y permite altas velocidades.

## 19. UPS
Sistema de respaldo eléctrico que mantiene encendidos los equipos cuando falla la energía.

## 20. PDU
Dispositivo que distribuye energía eléctrica a los equipos dentro de un rack.

## 21. CRAC
Aire acondicionado especializado para enfriar centros de datos.

## 22. Sensor ambiental
Dispositivo que monitorea temperatura, humedad o humo dentro del data center.

## 23. Redundancia
Uso de equipos o sistemas adicionales para evitar interrupciones.

## 24. N
Configuración donde solo existe la capacidad mínima necesaria para operar.

## 25. N+1
Configuración con un equipo extra de respaldo además de los necesarios.

## 26. 2N
Configuración donde todo el sistema está completamente duplicado.

## 27. 2N+1
Sistema totalmente duplicado con un respaldo adicional extra.

## 28. Colocation
Servicio donde una empresa renta espacio físico en un centro de datos externo.

## 29. Cloud
Uso de infraestructura y servicios tecnológicos a través de internet sin tener hardware propio.

## 30. Ruta de evacuación
Camino señalado para salir de forma segura en caso de emergencia.

## 31. Firewall
Sistema de seguridad que controla y filtra el tráfico de red.

## 32. Access Point
Dispositivo que proporciona conexión inalámbrica Wi-Fi.

## 33. Virtualización
Tecnología que permite ejecutar varias máquinas virtuales en un mismo servidor físico.

## 34. Máquina virtual
Sistema operativo virtual que funciona dentro de un servidor físico.

## 35. Backup
Copia de seguridad de información para recuperarla en caso de falla.

## 36. Monitoreo
Proceso de supervisar el estado de equipos, energía y temperatura.

## 37. Latencia
Tiempo que tarda la información en viajar de un punto a otro en la red.

## 38. Ancho de banda
Cantidad de datos que pueden transmitirse en una red en cierto tiempo.

## 39. Disponibilidad
Capacidad de un sistema para mantenerse funcionando y accesible.

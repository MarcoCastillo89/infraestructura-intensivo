## Lo que aprendí
A ver todos los procesos que hacen posible que los programas o aplicaciones se ejecuten de buena forma.
-  Procesos: Programas o aplicaciones en ejecución "Uso de CPU, Memoria Ram, Disco, Red, Consumo de energía"
-  Rendimiento: Uso de CPU, Procesador, Memoria Ram, Disco, Ethernet y GPU
-  Historial de aplicaciones: Medición de uso de recursos de una aplicación
-  Inicio: Aplicaciones que se ejecutan cuando se inicia Windows
-  usuarios: Usuarios que tienen una sesión activa en la computadora
-  Detalles: Es el mismo contenido que tiene procesos, pero más sencillo
-  Servicios: aplicaciones o servicios que se ejecutan en 2do plano

Tipos de errores que salen y su posible solución
-  CPU 100%
-  RAM LLENA
-  DISCO AL 100%
-  Procesos desconocidos
-  Se cierra solo
-  Access denied
-  Windows muy lento

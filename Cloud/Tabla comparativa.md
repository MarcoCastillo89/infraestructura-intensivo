## Tabla comparativa de Servicios

# Lab 1 — Mapeo de servicios Cloud

| Categoría | AWS | Azure | GCP | Cloudflare | Justificación de equivalencia |
|---|---|---|---|---|---|
| Cómputo | EC2 | Virtual Machines | Compute Engine | Workers | Todos permiten ejecutar aplicaciones y procesos en la nube. |
| Almacenamiento | S3 | Blob Storage | Cloud Storage | R2 | Todos almacenan archivos y datos de forma escalable y accesible desde internet. |
| Base de datos (BBDD) | RDS | Azure SQL Database | Cloud SQL | D1 | Son servicios administrados para guardar y consultar información en bases de datos. |
| CDN | CloudFront | Azure CDN | Cloud CDN | Cloudflare CDN | Distribuyen contenido desde servidores cercanos al usuario para mejorar velocidad y reducir latencia. |
| DNS | Route 53 | Azure DNS | Cloud DNS | Cloudflare DNS | Administran dominios y convierten nombres de dominio en direcciones IP. |
| Identidad y acceso | IAM | Entra ID (Azure AD) | Cloud IAM | Cloudflare Access | Controlan usuarios, permisos y autenticación en la nube. |
| Logs | CloudTrail | Azure Monitor Logs | Cloud Logging | Cloudflare Logs | Registran eventos y actividades para auditoría y diagnóstico. |
| Monitoreo | CloudWatch | Azure Monitor | Cloud Monitoring | Cloudflare Analytics | Supervisan rendimiento, tráfico y estado de los servicios. |
| Secretos | Secrets Manager | Key Vault | Secret Manager | Cloudflare Secrets | Guardan contraseñas, claves y datos sensibles de forma segura. |
| Mensajería | SQS | Service Bus | Pub/Sub | Queues | Permiten enviar y procesar mensajes entre aplicaciones y servicios. |
